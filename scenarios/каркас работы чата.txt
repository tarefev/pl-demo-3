
Сценарий выбора типа документа.
Завершение сценария:
  - Текст - "Тип документа выбран"
  - Выходим в состояние вольного ввода

определитель что происходит:

В каком состоянии находится чат:
A. Работает сценарий выбора типа документа
  A.1 Чоисы на выбор или свободный ввод
    1. игнорируем, предлагаем снова чоисы на выбор с поясняющим текстом
    2. продолжает сценарий
    3. (определитель)
      3.1 Если есть пилз с таким текстом - выбираем пилз
      3.2 Если текст подходит как название нового типа документа - устанавливаем новый тип документа
      3.3 Вольный ввод - игнорируем, предлагаем снова чоисы на выбор с поясняющим текстом
    4. игнорируем, предлагаем снова чоисы на выбор с поясняющим текстом
B. Работает подсценарий
  B.1 Предложил чоисы на выбор
    1. Спросить, прервать ли сценарий
      1.1 Если решили прервать, выполнить то, чего требуют данные согласно id
      1.2 Если решили не прерывать, предлагаем снова чоисы на выбор с поясняющим текстом
    2. Продолжение сценария
    3. (определитель)
      3.1 Если есть пилз с таким текстом - выбираем пилз
      3.2 Вольный ввод - Спросить, прервать ли сценарий
        3.2.1 Если решили прервать, ответить на вольный ввод
        3.2.2 Если решили не прерывать, предлагаем снова чоисы на выбор с поясняющим текстом
    4. Спросить, прервать ли сценарий
      4.1 Если решили прервать, запустить подсценарий "Свободый ввод" + Документ как вложение
      4.2 Если решили не прерывать, предлагаем снова чоисы на выбор с поясняющим текстом
  B.2 Ожидает текстового ввода
    1. Спросить, прервать ли сценарий
      1.1 Если решили прервать, выполнить то, чего требуют данные согласно id
      1.2 Если решили не прерывать, предлагаем снова чоисы на выбор с поясняющим текстом
    2. Выбирать нечего, сценарий недоступен
    3. (определитель)
      3.1 Если текст подходит под контекст, принимаем как ответ, продолжить сценарий
      3.2 Если текст не подходит под контекст - "Это был ответ или переформулировать или новый вопрос?" - 3 чоиса на выбор
        3.2.1 Если это был ответ, принять как ответ
        3.2.2 Переформулировать - Снова задать вопрос, ожидать свободный ввод
        3.2.3 Если это новый вопрос, ответить на новый вопрос, выйти из сценария без дополнительного подтверждения
    4. (определитель)
      4.1 Если документ подходит под контекст, принимаем как ответ, продолжить сценарий
      4.2 Если документ не подходит под контекст - "Документ не подходит под контекст"
        переходим на шаг B.2
C. Ни один сценарий не запущен, тип документа выбран
  1. Выполнить пилз из выпадайки-звездочки согласно id
  2. Выбирать нечего, сценарий недоступен
  3. Запустить подсценарий "Свободый ввод" (определитель)
    3.1 Если найден подсценарий (определитель)
      3.1.1 Уверенность высокая - запускаем сценарий
      3.2.1 Уверенность средняя или низкая - задаем уточняющий вопрос с пилзами
        3.2.1.1 - пилз выбран - запускаем сценарий
        3.2.1.2 - пользователь отменил запуск - пишем текст "Запуск не выполняем с доп вопросом", получаем состоние C
    3.2 Если это действие над документом - запускаем подсценарий "Действие над документом" (нужно расписать его (doc_ref, подтверждение на реализацию))
    3.3 Ответить на свободный ввод генерацией текста согласно контексту
  4. Запускаем подсценарий "Разбор файла"
D. Происходит разбор документа, запущены tools, чат думает, генерация текста - все это блокирует ввод, в т.ч. кнопок из ИИ-меню
  1. Выбирать нечего, сценарий недоступен
  2. Выбирать нечего, сценарий недоступен
  3. Выбирать нечего, сценарий недоступен
  4. Выбирать нечего, сценарий недоступен

1. прислали пилзы из выпадайки-звездочки - должен содержать id
2. выбрали некий чоис - должен содержать id
3. написали текст
4. приложили файл
